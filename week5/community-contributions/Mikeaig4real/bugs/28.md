## Correct Code

```python
def are_anagrams(str1, str2):
    if len(str1) != len(str2):
        return False
    str1_sorted = sorted(str1.replace(" ", "").lower())
    str2_sorted = sorted(str2.replace(" ", "").lower())
    return str1_sorted == str2_sorted
```
