## Correct Code

```python
def depth_first_search(graph, start_node):
    visited = set()
    stack = [start_node]
    result = []

    while stack:
        current_node = stack.pop()
        if current_node not in visited:
            visited.add(current_node)
            result.append(current_node)
            stack.extend(reversed(graph.get(current_node, [])))

    return result
```
