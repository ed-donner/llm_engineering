# Verifying References for Domestic Workers

References from previous employers help determine reliability and behavior.

## Recommended Reference Checks

Employers should confirm:

- Duration of employment
- Reason for leaving
- Behavior with children
- Reliability and punctuality

## Questions to Ask References

Examples:

- How long did the nanny work for you?
- Would you hire this nanny again?
- Did you ever have safety concerns?
- How did the nanny interact with children?

## Warning Signs During Reference Checks

- References refusing to speak
- Vague answers about performance
- Contradicting employment history

At least two independent references are recommended.
