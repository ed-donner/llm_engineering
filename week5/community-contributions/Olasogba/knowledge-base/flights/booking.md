## Booking Process

### Step-by-Step Guide

1. **Search Flights**
   - Select trip type (one-way, round-trip, multi-city)
   - Enter departure and arrival cities
   - Select dates and passenger count

2. **Select Flight**
   - Compare available flights
   - View fare options
   - Check seat availability

3. **Enter Passenger Details**
   - Full name (as on ID/passport)
   - Date of birth
   - Gender
   - Contact information
   - Frequent flyer number (optional)

4. **Add Extras (Optional)**
   - Seat selection
   - Extra baggage
   - Travel insurance
   - Lounge access
   - Airport transfers

5. **Payment**
   - Select payment method
   - Enter payment details
   - Confirm booking

6. **Confirmation**
   - Receive booking confirmation via email
   - Download e-ticket
   - Add to calendar
