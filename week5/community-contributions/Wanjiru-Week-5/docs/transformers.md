# Transformers and Large Language Models

Transformers are the neural network architecture used to build modern large language models such as GPT, Claude, and Llama.

Transformers process text using a mechanism called attention, which allows the model to understand relationships between words in a sequence.

Large language models trained with transformers can perform tasks such as:
- answering questions
- summarizing text
- generating code
- translating languages

However, language models do not have access to external knowledge by default. Techniques such as Retrieval Augmented Generation (RAG) allow them to access external documents and provide more accurate responses.
