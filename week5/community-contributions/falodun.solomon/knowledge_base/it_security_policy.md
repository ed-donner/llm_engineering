IT SECURITY POLICY

1. Purpose
This policy protects company systems and data from unauthorized access.

2. Password Requirements
Employees must use strong passwords and change them regularly.

3. Device Security
Company devices must be locked when unattended.

4. Data Protection
Sensitive company data must not be shared externally without authorization.

5. Software Installation
Employees must not install unauthorized software on company devices.

6. Reporting Incidents
Security incidents must be reported immediately to the IT department.
