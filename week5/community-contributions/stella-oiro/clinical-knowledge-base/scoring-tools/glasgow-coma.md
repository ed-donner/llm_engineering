# Glasgow Coma Scale (GCS)

## Components and Scoring

### Eye Opening (E)
| Score | Response |
|---|---|
| 4 | Spontaneous |
| 3 | To voice/verbal command |
| 2 | To pain |
| 1 | No response |

### Verbal Response (V)
| Score | Response |
|---|---|
| 5 | Oriented (knows person, place, time) |
| 4 | Confused (speaks in sentences, disorientated) |
| 3 | Inappropriate words (random, exclamatory) |
| 2 | Incomprehensible sounds (groaning, moaning) |
| 1 | No response |

### Motor Response (M)
| Score | Response |
|---|---|
| 6 | Obeys commands |
| 5 | Localises pain (moves hand toward stimulus) |
| 4 | Withdrawal from pain (flexion/normal) |
| 3 | Abnormal flexion (decorticate: arm flexion, wrist flexion) |
| 2 | Abnormal extension (decerebrate: arm extension, wrist pronation) |
| 1 | No response |

## Total Score Interpretation
**GCS = E + V + M (range 3–15)**

| GCS Total | Severity |
|---|---|
| 15 | Normal/fully conscious |
| 13–14 | Minor impairment |
| 9–12 | Moderate impairment |
| ≤ 8 | Severe (protect airway) |
| 3 | No function (deeply unconscious) |

**GCS ≤ 8: Intubate** — the airway is at risk; involve anaesthetics/ICU

## Paediatric GCS (for children < 2 years)
Verbal response modified:
- 5: Coos/babbles (normal)
- 4: Irritable cry
- 3: Cries to pain
- 2: Moans to pain
- 1: No response

## Limitations
- Cannot assess verbal score if intubated: document as "V: T (intubated)" and add T modifier
- Sedating medications lower score artificially — document if sedated
- Not a prognostic tool in isolation — use with clinical context

## Pupil Reactivity — AVPU Alternative
Quick bedside assessment:
- **A** — Alert
- **V** — responds to Voice
- **P** — responds to Pain
- **U** — Unresponsive

AVPU "P" ≈ GCS 8; AVPU "U" ≈ GCS 3

## Common Causes by Presentation
| GCS Pattern | Consider |
|---|---|
| Sudden drop + anisocoria | Intracranial haemorrhage, herniation |
| Gradual decline | Metabolic (hypoglycaemia, hepatic, uraemic) |
| Fluctuating | Seizures, toxic/alcohol |
| Focal deficit + reduced GCS | Stroke, mass lesion |

**Always check glucose** in any patient with reduced GCS (exclude hypoglycaemia before further investigation).
