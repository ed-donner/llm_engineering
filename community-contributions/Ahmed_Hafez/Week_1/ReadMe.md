# Inventory AI Agent - Ahmed Hafez

This project demonstrates an Agentic Workflow that analyzes inventory stock and generates strategic business reports.

## 📊 Dataset Instructions
To keep the repository lightweight, the dataset is hosted externally. 

1. **Download the Dataset:** [Click Here to Download Inventory_Dataset.csv](https://drive.google.com/file/d/16uOe1ZtjAri3thxCdBFZeNrJ9K1FthyF/view?usp=sharing)
2. **Setup:** Place the `Inventory_Dataset.csv` file in the same directory as the notebook.

## 🚀 How to Run
1. Open `Day5.ipynb` in Jupyter or VS Code.
2. Ensure you have your `OPENAI_API_KEY` configured.
3. Run the cells to see the Data Parser and Strategic Consultant agents in action.
