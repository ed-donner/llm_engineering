# HackTrack – an AI-Powered Hackathon Discovery Platform

## Background

HackTrack started as an extension of  the web scraping and local LLM exercises from week 1 of the AI Engineer Core Track course .

The original objective was to scrape information from a website and generate a summary using a local LLM. As I continued experimenting with the idea, it gradually evolved into a complete Flask-based web application for discovering and recommending hackathons.

## Features

* Aggregates hackathons from Devpost
* Stores hackathon data in MySQL (Hackathon details, User interests, etc)
* Supports keyword search
* Allows filtering by domain (Cybersecurity, AI, etc)
* Provides personalized recommendations based on user interests
* Generates AI-powered insights using a locally hosted Llama 3.2 model through Ollama

## Tech Stack

* Python
* Flask
* MySQL
* BeautifulSoup
* Ollama (Llama 3.2)
* HTML/CSS
