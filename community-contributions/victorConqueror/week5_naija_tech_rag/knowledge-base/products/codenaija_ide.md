# CodeNaija IDE

## Overview
CodeNaija IDE is NaijaCode's flagship product — a cloud-based integrated development environment built specifically for African developers. Launched in January 2022.

## Features
- **Cloud-Based Editor**: Write code directly in the browser with VS Code-level intelligence. Supports 20+ programming languages including Python, JavaScript, Go, Rust, and Solidity.
- **Real-Time Collaboration**: Multiple developers can code together in real-time, like Google Docs but for code. Powered by CRDT (Conflict-free Replicated Data Types).
- **Built-in Terminal**: Full Linux terminal access without needing a local development environment.
- **NaijaCode AI Assistant**: AI-powered code completion and suggestion engine trained on patterns common in African tech stacks (fintech, mobile money, USSD applications).
- **Low-Bandwidth Mode**: Optimized for African internet conditions — works smoothly on 2G/3G connections with offline caching.
- **Mobile Support**: Full IDE experience on Android devices, because many African developers code from their phones.

## Pricing
- **Free Tier (Omo Free)**: 3 projects, 1GB storage, community support
- **Pro Tier (Baller)**: ₦5,000/month — unlimited projects, 10GB storage, AI assistant, priority support
- **Team Tier (Squad)**: ₦15,000/user/month — everything in Pro + team management, shared workspaces, admin dashboard

## Tech Stack
- Frontend: React, TypeScript, Monaco Editor
- Backend: Go, Python (FastAPI)
- Infrastructure: AWS (Lagos region), Kubernetes
- Database: PostgreSQL, Redis
- Real-time: WebSockets, CRDT
