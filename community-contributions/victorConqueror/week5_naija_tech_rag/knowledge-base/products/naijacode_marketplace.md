# NaijaCode Marketplace

## Overview
NaijaCode Marketplace is a platform where developers can buy, sell, and share code templates, plugins, and complete project starters. Launched in June 2023.

## Features
- **Code Templates**: Pre-built project starters for common use cases — fintech apps, e-commerce sites, USSD applications, mobile money integrations.
- **Plugins**: Extensions for the CodeNaija IDE that add new functionality — linting rules, deployment shortcuts, database viewers.
- **Revenue Sharing**: Developers earn 80% of sales from their templates and plugins. NaijaCode takes a 20% platform fee.
- **Quality Review**: All submissions go through a review process to ensure code quality, security, and documentation standards.
- **African-First Content**: Templates specifically designed for African use cases — M-Pesa integration, Paystack/Flutterwave payment flows, MTN USSD apps, etc.
