# WEEK 7 Exercise: Price Prediction LLM Fine-Tuning

This assignment fine-tunes an open-source language model to predict the price of an item from its description using prompt-completion training.

The model is trained using QLoRA on the `ed-donner/items_prompts_lite` dataset.
