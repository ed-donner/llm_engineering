# AI Study Companion

An interactive AI tutoring assistant built for the **Week 2 LLM Engineering exercise**.
The application helps users understand **AI and machine learning concepts** through clear explanations generated by language models.

The interface is built with **Gradio**, supports **streaming responses**, and allows switching between different models while optionally using **Wikipedia as a reference tool**.

---

## Features

* **Interactive Gradio UI** – Ask questions directly in the browser
* **Streaming Responses** – Answers appear gradually as they are generated
* **Model Switching** – Choose between different models (OpenRouter or Ollama)
* **AI Tutor System Prompt** – Configures the model to explain concepts clearly
* **Wikipedia Lookup Tool** – Retrieves reference information to support explanations
* **Chat History** – Maintains context across multiple questions

---

## Example Questions

Try asking:

* *What are transformers in AI?*
* *Explain reinforcement learning in simple terms*
* *What are vector embeddings?*
* *How does attention work in transformers?*

## Setup

Install the required dependencies:

```bash
pip install gradio requests python-dotenv openai
```

Set your environment variables (example):

```bash
OPENROUTER_API_KEY=your_api_key
```

---

## Running the App

1. Open the Jupyter notebook.
2. Run all cells in order.
3. The notebook will print a **Gradio URL** such as:

```
http://127.0.0.1:7860
```

4. Open the link in your browser to interact with the AI Study Companion.
