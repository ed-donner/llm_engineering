# My Projects

## Project 1: SmartBudget
A personal finance tracker built with Django and React.
It uses ML to categorize transactions automatically and predict monthly spending.
Deployed on AWS with PostgreSQL database. Has 500+ active users.
Tech stack: Python, Django REST Framework, React, PostgreSQL, AWS EC2, S3.

## Project 2: CropWatch
An agricultural monitoring tool that uses satellite imagery and weather data.
Built for small-scale farmers in East Africa to predict crop yields and detect diseases.
Uses computer vision models (ResNet) trained on plant disease datasets.
Tech stack: Python, FastAPI, TensorFlow, OpenCV, Google Earth Engine API.

## Project 3: ChatDoc
A document Q&A chatbot that lets users upload PDFs and ask questions.
Uses LangChain, OpenAI embeddings, and FAISS for vector search.
Supports multi-document conversations with citation tracking.
Tech stack: Python, LangChain, Streamlit, FAISS, OpenAI API.
