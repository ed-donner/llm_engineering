# Week 8 Exercise: Multi-Agent Debate System for Price Band Classification

## Deliberative Multi-Agent Architecture: Debate + Judge Crew

This notebook implements a multi-agent system for price band classification, featuring:
- Advocate Agents (Budget, Midrange, Premium) with specialized strategies
- Judge Agent with rigorous evaluation rubric
- Interactive UI for real-time debate visualization

### Architecture
- **BudgetAdvocateAgent**: Identifies value products and budget signals
- **MidrangeAdvocateAgent**: Expert in mainstream market positioning
- **PremiumAdvocateAgent**: Specialist in luxury/high-end indicators
- **JudgeAgent**: Impartial evaluator with systematic scoring rubric

### Key Features
1. Specialized Advocate Agents with unique decision-making strategies
2. Rigorous Judge Evaluation (5-dimension scoring)
3. Deliberative Process (evidence-based, honest, transparent)
4. Interactive UI (color-coded, real-time, scoring breakdown)
