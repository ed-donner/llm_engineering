Prompt Cost Calculator
A simple Gradio app to estimate and compare LLM API costs before sending a prompt.
It calculates input/output tokens and shows projected costs across multiple AI models (OpenAI, Anthropic, Google, etc.), sorted from cheapest to most expensive.
 
Features
Token counting using tiktoken
Output length estimation (Short, Medium, Long, Code)
Monthly usage scaling
Cost comparison table
Cheapest vs most expensive model summary
Simple visual cost breakdown
